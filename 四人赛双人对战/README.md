# 配置信息说明
**脚本默认不为增强版**，非增强版利用本地ocr识别题目，在识别速度、识别正确率上比第三方ocr差。此脚本选用华为API或者百度API实现OCR功能，如果你想使用增强版，选择其中一个配置即可
